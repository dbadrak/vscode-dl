# Welcome Bienvenue sur le mirroir interne de [Visual Studio Code](http://code.visualstudio.com). ## Installation 1. Télécharger le [paquet Debian](@@CODE_URL@@) et taper dans une console : ``` $ sudo dpkg -i @@CODE_DEB@@ ``` 2. Désactiver les mises à jour automatiques et la télémétrie dans les [Settings](https://code.visualstudio.com/docs/getstarted/settings). ``` { "update.channel": "none", "telemetry.enableTelemetry": false, "telemetry.enableCrashReporter": false, "extensions.autoUpdate": false } ``` 3. Installer les extensions
## Extensions Pour installer une extension : - télécharger le fichier `.vsix` en cliquant sur le lien de la colonne **Version** - taper dans une console la commande : `code --install-extension .vsix` **ATTENTION** si vous installez l'extension C/C++, vous devez obligatoirement installer l'extension `cpptools-linux` correspondante _après_.
