# Welcome Bienvenue sur le mirroir interne de [Visual Studio Code](http://code.visualstudio.com). ## Installation 1. Télécharger le [paquet](code_1.24.0-1528306776_amd64.deb) et taper dans une console : ``` $ sudo dpkg -i code_1.24.0-1528306776_amd64.deb ``` 2. Désactiver les mises à jour automatiques et la télémétrie dans les [Settings](https://code.visualstudio.com/docs/getstarted/settings). ``` { "update.channel": "none", "telemetry.enableTelemetry": false, "telemetry.enableCrashReporter": false, "extensions.autoUpdate": false } ``` 3. Installer les extensions **ATTENTION** si vous installez, l'extension C/C++, vous devez aussi installer l'extension `cpptools-linux` _après_. ## Extensions
